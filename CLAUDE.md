# ReadLater

Personal Android reader for software-development news and evergreen engineering
articles. It combines Hacker News and RSS, supports bookmarks and topic
preferences, and ships an offline Classics pool.

## Agent instructions

Read and follow [`AGENTS.md`](AGENTS.md). It is the operational authority for
all agents and tools.

## Quick commands

```bash
./gradlew :app:testDebugUnitTest
./gradlew :app:assembleDebug
./gradlew :app:connectedDebugAndroidTest
```
