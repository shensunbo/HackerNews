retained for Git history and design archaeology, not as default instructions for
current work.

| File | Use it when |
| --- | --- |
| `spec.md` | Investigating the original product request. |
| `design.md` | Tracing an intentional original product decision. |
| `ui-design.md` | Comparing a UI change with the original terminal visual system. |
| `implementation-plan.md` | Looking up original task sequencing or commands. |
| `plan-review.md` | Checking why an original P0/P1/P2 correction exists. |
| `execution-status.md` | Auditing historical verification evidence for its recorded commit only. |

For active work, use [`CURRENT_STATE.md`](../../CURRENT_STATE.md) and
