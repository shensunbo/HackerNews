# goal
制作一个新闻推送app，帮我推送关于软件开发的新闻资讯博客等等。

# 方案
1. 可以只提供链接和简介，跳转到第三方浏览器。比如chrome等来阅读完整内容。

# 设备
我的设备是安卓手机，系统为android 15。已经连接到设备上了，你可以获取你需要的信息。这个是vivo X100S PRO，这个apk只需要在这个手机上使用，不会上架应用商店。

# 其他
1. 需要保证软件的可拓展性，比如可以配置侧重不同的topic等，但是都是与软件开发相关的。
2. 不一定是最新的新闻和最近的新闻，经典的也可以，但是是需要有用的，作为软件开发者应该知道的。
3. 面向的是初级和中级的软件开发者，内容需要有一定的深度和实用性，但不需要过于复杂。
